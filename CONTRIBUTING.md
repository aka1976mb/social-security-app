## ğŸ›  Development Setup

### Prerequisites
- Node.js (v18+) or Python (3.10+)
- PostgreSQL
- Git

### Steps
1. Clone the repo:
   ```bash
   git clone https://github.com/aka1976mb/social-security-app.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
   or  
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app locally:
   ```bash
   npm run dev
   ```
   or  
   ```bash
   python manage.py runserver
   ```
